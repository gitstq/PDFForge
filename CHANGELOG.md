# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-05-19

### Added
- Initial release
- Zero-dependency PDF parsing engine
- Text extraction capabilities
- Table detection and extraction (CSV/JSON)
- Document structure analysis
- Markdown conversion
- Keyword extraction
- Batch processing support
- Multi-language documentation (EN, ZH-CN, ZH-TW, JA)
- Beautiful CLI interface with colors
- Comprehensive test suite
- MIT License

### Features
- 📄 **Text Extraction**: Extract text from entire PDF or specific pages
- 📊 **Table Detection**: Auto-detect and export tables
- 🏗️ **Structure Analysis**: Analyze headings, paragraphs, lists
- 📝 **Markdown Export**: Convert PDF to Markdown format
- 🔑 **Keyword Extraction**: Extract keywords using frequency analysis
- 📦 **Batch Processing**: Process multiple PDFs at once
- 💾 **JSON Export**: Export PDF data with metadata
- 🎨 **Colorful CLI**: Beautiful, informative output

### Supported Platforms
- Windows
- macOS
- Linux

### Requirements
- Python 3.8+
- No external dependencies (zero dependency design)
